SQL Judges Your Life 💀

An AI-powered interactive SQL analytics experience built with React + Claude API.

🚀 Overview

SQL Judges Your Life is a frontend-driven analytical simulation that transforms structured user inputs into dynamically generated SQL queries and derived metrics.

Users enter real behavioral data (food habits, gym attendance, sleep cycle, spending patterns, text response behavior), and the system generates:

A realistic analytical SQL query

Computed statistics derived from their inputs

A data-driven verdict

Highlighted SQL syntax

The project simulates how real analytical SQL pipelines work:

User Input → Structured Prompt → Generated SQL → Parsed JSON → Rendered Metrics
🧠 What This Project Demonstrates

Analytical SQL query design thinking

Structured data modeling via dynamic field schemas

AI-powered structured output generation

Strict JSON schema enforcement

Regex-based SQL syntax tokenization

Frontend state-driven architecture

API integration & response parsing

This is not just a “fun app” — it is a simulation of analytical workloads using AI as a query generation engine.
